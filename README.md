Todo List App

This is a simple Todo List application built with Angular.
The project demonstrates basic task management functionality with a clean and minimal UI.

Features
Add new tasks
Mark tasks as completed
Delete tasks
Simple and responsive interface
Basic state management on the client side

Tech Stack
Frontend: Angular
TypeScript
HTML / SCSS
Fake placeholder database 


This project was created for learning and practicing Angular fundamentals, including components, data binding, and basic UI logic.
